### Uploading Images

**Step by step**:
1. Find any field with an image icon or "Upload" button
2. Click it

### Image Recommendations

**For best results**:

```
Featured Images (blog posts):
  - Size: 1200 x 630 pixels
  - Format: JPG
  - Max file size: 500KB

Testimonial Photos:
  - Size: 200 x 200 pixels
  - Format: JPG or PNG
  - Max file size: 100KB

General Images:
  - Keep under 1MB
  - Use JPG for photos
  - Use PNG for logos/graphics
  - Optimize before upload (use TinyPNG.com)
```

## ğŸ’¾ Save vs Publish

### **Save** (Draft mode)
- Saves your changes
- Changes NOT visible on website yet
- You can come back later to edit more
- Good for: Work in progress

### **Publish**
- Makes changes live on website
- Triggers automatic deployment
- Takes 30-60 seconds to go live
- Good for: Final changes ready to show

## ğŸ†˜ Common Issues

### "Save" button grayed out
- You haven't made any changes yet
- Or you have validation errors (check for red error messages)

### Changes not showing on website
- Wait 60 seconds after publishing
- Hard refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
- Check Netlify dashboard for deploy status

### Can't upload image
- File too large (max 10MB, but keep under 1MB recommended)
- Wrong file format (use JPG, PNG, or GIF)
- Slow internet connection - wait and try again

### Lost my work
- Admin auto-saves drafts every few minutes
- Check "Workflow" tab for unsaved drafts
- Next time: Save more often!
